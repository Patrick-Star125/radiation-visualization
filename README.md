# radiation-visualization

这是一个赚钱的外包项目，主要是做一个软件带有客户端和模块化天线的各项指标可视化功能

大致架构如下

![](http://1.14.100.228:8002/images/2022/01/10/20220110225435.png)

## 项目文件结构

**client**

客户端界面代码，包括开始界面、主界面等

**documentation**

项目文档

**draw**

画图模块代码

**utils**

工具代码
